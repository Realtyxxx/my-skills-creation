# Ada

## SM Targets

- SM 89

## Notable features

- 4th gen Tensor Cores with FP8 support
- 3rd gen RT Cores
- Shader Execution Reordering

## Compile notes

- Use Ada targets for RTX 40-series or RTX 6000 Ada generation requests.
- Ada shares some high-level capabilities with Hopper, but Hopper-only features
  such as Thread Block Clusters should stay out of this page.
